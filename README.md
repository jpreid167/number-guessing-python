# Python Number Guessing Game

This is a console-based number guessing game written in Python. The program randomly selects a number within a user-defined range, and the player tries to guess it.

## Features
- Custom number range
- Unlimited guesses
- Feedback for guesses (too high / too low)
- Input validation
- Option to quit at any time

## Technologies Used
- Python
- Random module
